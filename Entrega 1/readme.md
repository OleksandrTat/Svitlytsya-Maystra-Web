# Svitlytsya Maystra - MVP (Etapa 1)

Proyecto web para un taller creativo que centraliza la captacion de solicitudes, muestra un catalogo de trabajos y ofrece un panel admin basico para gestionar contenidos.

## 📖 Explicacion idea del proyecto
Crear una plataforma moderna y sencilla que conecte a clientes con el taller, permita explorar trabajos y servicios, y habilite una gestion interna rapida de solicitudes y contenidos.

## 🎯 Nuestros Objetivos
- Validar el MVP con una presencia web clara y funcional.
- Centralizar la recepcion de solicitudes de clientes.
- Mostrar el catalogo de trabajos y servicios de forma ordenada.
- Facilitar la gestion interna mediante un panel admin.
- Medir el uso con analitica para iterar el producto.

## 🌐 Tecnologias Usadas
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Supabase (PostgreSQL, Auth, Storage, RLS)
- Resend (correo transaccional)
- PostHog (analytics)

## 🔗 Funcionalidades Web
- Landing con propuesta de valor.
- Catalogo de trabajos con paginas de detalle.
- Seccion de servicios.
- Formulario de contacto para captacion de leads.
- Panel admin para gestionar contenido basico.
- Autenticacion de administradores.
- Notificaciones por email para nuevas solicitudes.
- Paginas legales (privacidad, terminos, cookies).
